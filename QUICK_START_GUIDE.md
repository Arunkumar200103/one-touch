# One Touch - Quick Start Guide

## Website Structure

### Main Pages
- **Home** (`/`) - Dashboard with categories and popular services
- **How It Works** (`/how-it-works`) - Step-by-step tutorial
- **Service Inquiry** (`/service-inquiry`) - Multi-step service request form
- **About** (`/about`) - Company mission and values
- **Reviews** (`/reviews`) - All customer reviews with filters
- **Favorites** (`/favorites`) - Bookmarked businesses
- **Safety** (`/safety`) - Verification and safety information
- **FAQs** (`/faqs`) - Searchable questions and answers
- **Contact** (`/contact`) - Support contact options

### Dynamic Pages
- **Category** (`/category/[name]`) - Service listings by category
  - Shows 4 service types + verified businesses
  - Search and filter options
  - Back to home button
- **Business Profile** (`/business/[id]`) - Detailed business information
  - Photos, ratings, reviews
  - Contact methods (phone, WhatsApp, email)
  - Services offered
  - Experience and background

## Getting Started

### To View a Category
1. Click a category card on home page
2. See service types section
3. Browse verified businesses
4. Search for specific services

### To Request a Service
1. Go to "Service Inquiry"
2. Select service type
3. Enter your details
4. Choose appointment
5. Submit request

### To Leave a Review
1. Go to "Reviews"
2. Filter by business/category
3. Sort by rating
4. Read customer feedback

### To Save Favorites
1. Browse businesses
2. Click heart/bookmark icon (feature ready)
3. View in "Favorites" page
4. Compare multiple businesses
